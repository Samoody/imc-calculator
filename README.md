# 💻 Calculadora de IMC | JavaScript

Projeto desenvolvido com foco em prática de lógica de programação, manipulação do DOM e experiência do usuário.

---

## 🚀 Funcionalidades

- Inserção de nome, idade, peso e altura  
- Cálculo automático do IMC  
- Classificação do IMC (abaixo do peso, normal, sobrepeso, obesidade)  
- Recomendações de saúde baseadas no resultado  
- Sistema de histórico (salvo no navegador)  
- Limpar dados e histórico  
- Interface responsiva (funciona no celular 📱)  
- Feedback visual e animações  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## 🎯 Objetivo

Este projeto foi criado para treinar:

- Lógica de programação  
- Manipulação do DOM  
- Validação de dados  
- Armazenamento local (localStorage)  
- Organização de código  

---

## ▶️ Como usar

1. Abra o arquivo `index.html`
2. Preencha os campos com seus dados
3. Clique em **Calcular**
4. Veja seu resultado e recomendações
